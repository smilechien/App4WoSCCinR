# -App4WoSCCinR

A web-based application for streamlined bibliometric analysis using Web of Science Core Collection (WoSCC) metadata.

Overview

App4WoSCCinR enables users to perform bibliometric analysis directly from standard 72-column WoSCC Excel files without requiring programming skills. The platform integrates preprocessing, performance reporting, and advanced visualization into a single workflow.

Key Features
Upload WoSCC .xls/.xlsx files or load via URL
Automated data cleaning and normalization
Summary report across 10 metadata domains
100+ visualizations (e.g., slope, Sankey, chord, SSplot, maps)
Dominance analysis using Absolute Advantage Coefficient (AAC)
Export results as PNG or ZIP for reproducibility
Workflow
Upload WoSCC metadata file or input URL
Click Run WoS to generate analysis and visualizations
Input Requirements
WoSCC export in 72-column Excel format (.xls/.xlsx)
Optional: .csv or .txt via URL
Outputs
Summary performance report (n, h, AAC)
Multi-domain visualizations
Downloadable figures (PNG) and full results (ZIP)
Use Case

Designed for researchers, educators, and analysts seeking a transparent, efficient, and reproducible bibliometric workflow without reliance on specialized software.
